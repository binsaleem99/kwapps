# KWQ8 Beta Testing Program
**Duration:** 2 weeks | **Target:** 20 testers

## Beta Tester Recruitment

### Target Profile
- **Location:** GCC countries (Kuwait, Saudi, UAE, Qatar, Bahrain, Oman)
- **Language:** Arabic speakers
- **Business:** Small business owners or tech-savvy individuals
- **Industries:** Mix of restaurant, salon, retail, services, portfolio

### Recruitment Sources
1. **Kuwait Tech Hub** - Startup community
2. **Instagram** - Arabic business accounts (500-5K followers)
3. **WhatsApp Business Groups** - Kuwait/GCC entrepreneurs
4. **Existing Test Group** - Previous testers
5. **Referrals** - Ask beta testers to invite 1 friend

## Beta Testing Tasks

### Week 1: Core Features

**Day 1-2: Template System**
- [ ] Browse template gallery
- [ ] Select a template matching your business
- [ ] Complete 5-step customization wizard
- [ ] Generate website
- [ ] Rate experience (1-5)

**Day 3-4: Visual Editor**
- [ ] Open visual editor
- [ ] Make 3-5 edits via Arabic chat
- [ ] Test undo/redo
- [ ] Upload images
- [ ] Test device preview (desktop/mobile)

**Day 5-7: Admin Dashboard**
- [ ] Access auto-generated admin dashboard
- [ ] Add 3+ products/services
- [ ] Upload images
- [ ] Test if changes appear on site
- [ ] Rate admin dashboard (1-5)

### Week 2: Publishing & Payments

**Day 8-10: Domain & Publishing**
- [ ] Search for domain
- [ ] Purchase domain (or use free domain if eligible)
- [ ] Deploy website
- [ ] Verify custom domain works
- [ ] Check SSL certificate

**Day 11-12: WhatsApp Integration**
- [ ] Configure WhatsApp widget
- [ ] Test widget appears on site
- [ ] Test clicking widget opens WhatsApp
- [ ] Verify pre-filled message works

**Day 13-14: Payment Flow (Optional)**
- [ ] Complete actual subscription (1 KWD trial)
- [ ] Test credit system
- [ ] Test daily bonus credits
- [ ] Cancel subscription (test cancellation flow)

- Add to post-launch roadmap
- Communicate timeline to beta testers

- Get permission for website use
