# KWq8.com Claude Code Agents

**Springwood | Kuwait | December 2025**

## Quick Setup

1. Copy the `claude-agents` folder to your project as `.claude/agents/`:
   ```bash
   cp -r claude-agents /path/to/kwq8/.claude/agents
   ```

2. Copy `mcp-config.json` to your Claude Code settings

3. Open Claude Code in your project and use the agents:
   ```
   @orchestrator - Coordinate development
   @frontend-builder - UI development
   @backend-builder - API & database
   @gemini-logic - AI pipeline
   @qa-tester - Testing
   @code-reviewer - Code review
   ```

## Agents Included

| Agent | Purpose |
|-------|---------|
| orchestrator.md | Chief coordinator |
| frontend-builder.md | React, Tailwind, RTL |
| backend-builder.md | Supabase, APIs, Payments |
| gemini-logic.md | AI pipeline design |
| qa-tester.md | Playwright, testing |
| code-reviewer.md | Security, quality |

## MCP Servers Required

- Supabase MCP
- Playwright MCP
- GitHub MCP
- n8n MCP

Install with: `npm install -g supabase-mcp @anthropic-ai/mcp-playwright`
